CC CRM
======

Description
-----------

Customer Relationship Management for company "Constructive Council". The customer needed a simple, project-task oriented system specifically optimal for mobile devices.

Developed with:
* back-end: PHP/ MySQL
* front-end: HTML5/JS (Bootstrap/ jQuery)

Logo
----

The logo had to represent industrial feeling offering trust, performance and security.
